Steps to follow:
1.Create Data base by run the Sql script available in SignrDB.sql file
2.Set connection string in SignalRBackend project.
3.Run both signRfrontend and SignRBackEnd projects
4.In login screen give UserName and login
5.Open app in another browser window also 
6.If you send message from one login ,it will send In-app notification from backend. 
7.You can see the notifications count in notify icon and you can also view the list of notifications under notify icon.
8.If you need, you can add more methods in Hub and receiving notifications in FrontEnd Project by creating Js event in chat.js file.
